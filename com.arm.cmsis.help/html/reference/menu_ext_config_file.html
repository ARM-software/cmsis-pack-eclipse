Configuration File Context Menu

Configuration Wizard

Files that have CMSIS Configuration annotations can be opened with a GUI-like editor. Right-click the file and select:

  • Open With → CMSIS Configuration Wizard
File Modifications

The Project Explorer indicates changes of Software Component configuration files with the following icons:

  • file differences.
  • folder differences.

Right-click on the Software Component file and select one of the following:

  • CMSIS C/C++ Project → Upgrade to copy a newer version of the configuration file to the project.
  • CMSIS C/C++ Project → Downgrade to copy an older version of the configuration file to the project.
  • CMSIS C/C++ Project → Merge to open a file compare that allows you to merge previous settings into the new configuration file.
