CMSIS Pack Manager - Boards

The Boards view shows all boards that are supported by software packs, and serves as a filter to show relevant software packs and example projects. The Summary field shows the amount of devices that match your search criteria and provides documentation on board and device level.

Expand the list or enter the name of a board using wildcards into the field Search Board to narrow the list.
