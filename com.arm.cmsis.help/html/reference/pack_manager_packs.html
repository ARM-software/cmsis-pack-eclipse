CMSIS-Pack Manager - Packs

The Packs view lists all available packs on the Internet or workstation that match your search criteria.

You can narrow the list by:

  • Selecting a device in the Devices view.
  • Selecting a board in the Boards view.
  • Typing the pack name using wildcards in Search Pack.

Expand a selected pack to show all installed or previous versions.

Hover over a pack name to display brief revision information.

Execute an Action for a software pack:

Install the software pack. Download the package from the Internet and install the software.

Update the software pack. A newer version is available on the Internet or the workstation. Projects that use Software Components from this package get updated.

Install the software pack. The package has been downloaded already, but is not installed yet.

Remove the installed software pack and example projects. The software pack itself remains in the download area of the folder ..\ARM\Pack and can be reinstalled if needed. Removed Software Components that are used explicitly in projects get flagged as missing.


The installed package is up to date. No further action required.

This software pack is not maintained any longer. Hover the mouse over the button to view the recommended software pack. Could also identify that you have no connection to the network.

This software pack is not stored in the pack root folder, but is used from a local repository. Refer to Using Local Repositories for more information.

This software pack has been deprecated. The color of the icon reflects the current installation state. Deprecated software packs can still be installed. Expand the software pack to select the version. The tooltip of a deprecated package can display the date of deprecation and a potential replacement package.

Delete software packs that could not be read due to errors in the file. Hover the mouse over the button to view details. The button can remove files or entire folders. You may have to check for updates to reload software packs.

Delete all software packs marked with the Delete button. Hover the mouse over the button to view details. The button can remove files or entire folders.
