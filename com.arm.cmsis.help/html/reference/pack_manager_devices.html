CMSIS Pack Manager - Devices

The Devices view serves as a filter to find relevant software packs and example projects. The Summary field shows the amount of devices that match your search criteria. Access to documentation is provided on device level with an hyperlink.

Expand the list or enter the name of a device using wildcards into the field Search Device to narrow the list.
