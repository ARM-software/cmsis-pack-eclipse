CMSIS Pack Manager - Examples

The Examples view shows example projects for the selected device or board that are available in software packs on the workstation.

Enter an example name using wildcards in the field Search Example to narrow the list.

Disable Only show examples from installed Packs to show all examples from software packs on your workstation.

Hover over a pack name to display brief information.

Execute an Action for the example project:

Install the software pack that contains the example. It may initiate a download of the software pack from the Internet.
opens a pop-up dialog showing example information. In case the example project exists already in the workspace, user can replace or make another copy of the project.
