RTE Configuration

The *.rteconfig file of the project stores:

  • Software Components that are selected from Software Packs.
  • Selected Device of the target device and processor.
  • Software Packs that are selected for the project.

When you create a project or open an existing *.rteconfig file, the RTE configuration editor opens.

Save the *.rteconfig file to apply project modifications.
