CMSIS C/C++ Project

A CMSIS C/C++ Project uses the information from Software Packs to setup device related tool parameters. You may select also pre-build software components from Software Packs to create your application code.

Use the following menu commands:

  • File → New → Project and select C/C++ → C Project.
  • In the C Project dialog, enter the project name and select CMSIS C/C++ Project. Select a toolchain of your choice, for example the ARM Compiler.
  • The Select Device dialog shows the device support available from the Software Packs that are installed on your computer. Choose the device for your project, for example the i.MX 7 Dual – MCIMX7D:Cortex-M4 processor.
  • The Select Configurations dialog allows you to create different configurations of your project.

The *.rteconfig file allows you to configure the Run-Time Environment (RTE), for example, add software components to your project.

The next steps to create a project are:

  • Add Software Components as described in RTE Configuration - Components.
  • Add template files to the project workspace.

The CMSIS C/C++ Project manager uses the CDT standard functionality to configure project properties or control the build process.

See also:

  • RTE Configuration to change device and select software components.
  • Menu Extensions to reset project properties or remove unused files.
