Using Local Repositories

All versions of installed packs are stored below the pack root folder (refer to CMSIS Pack Manager). While developing a software pack, it is useful to have direct access to its content without the need to re-build and re-install the pack after modifications. For this purpose, you can add the folder of a pack's local copy by selecting Manage Local Repositories... and pressing Add.... Select the PDSC file in the folder and press OK:

Using the Remove button, you can remove local repositories from the list.

The plugin does not make a difference between local repositories and installed packs. If a version of a pack is provided as an installed pack and a local repository, only the content from the local repository is used.

Changes to the PDSC file of a local repository are not notified to running Eclipse instances. To reload the PDSC file, use the Reload Packs button.
